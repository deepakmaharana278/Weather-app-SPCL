# 🌤️ Weather Forecast App

A beautiful, responsive weather application built with React and Vite that provides real-time weather data and 5-day forecasts for cities worldwide.

## 🚀 Live Demo
[View Live Demo](https://deepak-weather.netlify.app/) - Click to see the app in action!


## ✨ Features

- **📍 Live Location Detection** - Automatically detects and shows weather for your current location
- **🔍 City Search** - Search for any city worldwide
- **🌡️ Current Weather** - Temperature, feels like, humidity, wind speed, pressure
- **📅 5-Day Forecast** - Daily weather predictions with icons and min/max temperatures
- **🌬️ Air Quality Index** - Real-time AQI data with PM2.5 levels
- **🌅 Sunrise/Sunset** - Precise timings for your location
- **💾 Recent Searches** - Saves last 5 searched cities with clear option
- **📱 Fully Responsive** - Works on mobile, tablet, and desktop

## 🛠️ Technologies Used

- **React 18** - UI library
- **Vite** - Build tool and development server
- **Tailwind CSS** - Styling and responsive design
- **React Icon** - Weather icons and UI elements
- **OpenWeatherMap API** - Weather data provider

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- OpenWeatherMap API key ([Get one here](https://openweathermap.org/api))

## 🚀 Installation

```
# Clone the repository
git clone https://github.com/yourusername/weather-app.git

# Navigate to project directory
cd weather-app

# Install dependencies
npm install

# Start development server
npm run dev
```
